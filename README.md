# 🎯 HaxBall Aim Assist

Mira de trajetória em tempo real para HaxBall. Funciona 100% online, sem modificar o servidor.

## Instalação

### 1. Instalar Tampermonkey
- [Chrome / Edge](https://www.tampermonkey.net/)

### 2. Instalar o userscript
Clique no link abaixo para instalar direto no Tampermonkey:

👉 **[Instalar HaxBall Aim Assist](https://raw.githubusercontent.com/brunobm7/haxball-aim/main/haxball-aim.user.js)**

### 3. Jogar
Abra qualquer sala no HaxBall. O painel de calibração aparece automaticamente no canto superior direito.

---

## Como calibrar

1. Clique em **"Marcar Jogador"** → clique sobre seu personagem no jogo
2. Clique em **"Marcar Bola"** → clique sobre a bola
3. A mira ativa automaticamente quando você está perto da bola

A calibração é salva e não precisa repetir a cada sessão.

---

## Atalhos

| Tecla | Ação |
|-------|------|
| `M` | Ativar / desativar mira |
| `C` | Abrir modo calibração |
| `+` / `-` | Comprimento da mira |
| `[` / `]` | Opacidade |
| `Esc` | Cancelar calibração |
